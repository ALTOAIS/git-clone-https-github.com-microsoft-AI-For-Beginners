# Phase 2A — Risk register

Carried forward from the Phase 2A architecture report, updated to
reflect the corrections accepted in this documentation pass (notably:
the `GrammarRuleResolver` fix removes the "wrong rule selected" failure
mode that a naive `LIMIT 1` lookup would have created; Git-backed
versioning removes the "can't actually roll back" gap the original
report's plain `version` integer left open).

| Risk | Probability | Impact | Early warning | Mitigation | Owner/workstream | Blocking? |
