# Семейный квиз (Family Quiz Night)

Веб-приложение для проведения семейного квиза на телевизоре или большом экране
(по PRD «Family Quiz Night Platform»). Две команды — **Парни** и **Девушки**,
игрой управляет один ведущий с ноутбука. Без сервера, без интернета, без регистрации.

## Запуск

```bash
npm install
npm run dev        # режим разработки: http://localhost:5173
npm run build      # сборка в dist/ (можно открывать без сервера)
npm run preview    # локальный просмотр сборки
```

## Структура игры

| Тур | Формат | Подсчёт |
|---|---|---|
| 1. Разминка | 12 вопросов × 4 варианта, таймер 30 сек, ответы после всех вопросов | ведущий вводит число правильных ответов |
| 2. Где логика? | 5 вопросов, по три картинки | ведущий вводит число правильных ответов |
| 3. Чёрный ящик | подсказки открываются по одной, затем ответ | ведущий выбирает победившую команду |
| 4. Битва полов | 10 вопросов парням + 10 девушкам, ответы после каждого блока | ведущий вводит результат каждого блока |
| 5. Финал: Ближе всех | 8 вопросов без вариантов, ставка 1/2/3 перед вопросом | победитель получает ставку, проигравший теряет свою |

Счёт считается автоматически, сохраняется в LocalStorage (игру можно продолжить
после перезагрузки страницы), после каждого тура показывается промежуточная
таблица, после финала — итоговая таблица, победитель и анимация победы.

С момента старта игры идёт скрытый секундомер ведущего: в верхней панели видна
только иконка ⏱, время показывается по клику (и снова скрывается повторным
кликом) — удобно для суперфинала «сколько минут идёт квиз». Время старта
сохраняется вместе с игрой и переживает перезагрузку страницы.

## Как поменять вопросы

Весь контент игры лежит в одном файле — **`src/data/quiz_data.json`**.
Меняются только данные, код приложения трогать не нужно.

- Вопросы туров 1 и 4 могут быть с вариантами (`options` + `correctIndex`)
  или открытыми (только `answer`).
- «Картинки» тура 2 — это эмодзи или пути/URL файлов изображений
  (`"images": ["🍞", "./img/photo.png", "https://…/pic.jpg"]`).
- Черных ящиков может быть несколько (`items`), у каждого свои подсказки,
  ответ и цена победы (`points`).

## TV First

Интерфейс полностью на русском языке, корректно отображает казахские буквы
(Ә Ғ Қ Ң Ө Ұ Ү Һ І), рассчитан на просмотр с 3–5 метров: крупные шрифты и
кнопки, высокая контрастность, кнопка полноэкранного режима.

## Технологии

React 19 · TypeScript · Vite · LocalStorage. Backend отсутствует —
всё работает в одном браузере.
